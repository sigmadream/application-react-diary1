# React 연습을 위한 `일기` 웹 어플리케이션입니다.

> 해당 저장소는 `React`를 연습하기 위한 저장소입니다. 백엔드 개발자들이 `React`가 어떻게 작동하는지 확인하기 위해서 만들어둔 저장소입니다. `React`를 사용하여 `일기`를 작성하고, 저장하고, 수정하고, 삭제하고, 검색하고, 공유하는 웹 어플리케이션입니다.

## 설정

```bash
$ npm install
$ npm run start
```

## 기능

- [x] 일기 입력
- [x] 일기 저장
- [x] 일기 수정
- [x] 일기 삭제
- [x] 일기 목록
- [ ] 일기 검색
- [ ] 일기 상세
- [ ] 일기 공유
